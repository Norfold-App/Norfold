# Contributing

Thanks for your interest — but **Norfold does not accept external
contributions** at this time.

This repository is source-available for viewing only (see
[LICENSE](LICENSE) and [NOTICE.md](NOTICE.md)). Pull requests will be
closed without review. Per Section 4 of the license, any material you
submit anyway is assigned to the copyright holder.

## Bug reports

You're welcome to open a **GitHub issue** to report a bug or security
concern in the published code. Please do not include exploit details
for security issues in public issues — say it's security-related and
the maintainer will follow up.

## Questions / licensing

For anything else, including requests to use the code under different
terms, open an issue.
