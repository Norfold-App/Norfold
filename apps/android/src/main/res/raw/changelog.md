# Norfold 0.1.0

## Workspace completion

- Rebuilt onboarding as a resumable ten-step local-first flow.
- Consolidated Workspace, Permissions, Backup, Import, Sync, and Conflict settings.
- Added adaptive task Table, Board, List, Feed, Calendar, Chart, Gallery, Timeline, and Matrix views.
- Added column-backed statuses, focused property editors, checklist progress, task date ranges, comments, and files.
- Improved note Page and Preview editing, adaptive navigation, encrypted backup, and local diagnostics.

Norfold is currently an alpha release. Cloud controls appear only when their public configuration is present.
